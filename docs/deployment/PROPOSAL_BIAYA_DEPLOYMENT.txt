## 1. PENDAHULUAN
Sehubungan dengan rencana peluncuran (launching) website resmi **Mahira Tour**, kami memerlukan penyediaan infrastruktur server (VPS) dan domain agar layanan dapat diakses secara publik oleh calon jamaah. 

Kami merekomendasikan penggunaan spesifikasi server yang stabil (AMD eXtreme) guna mendukung fitur sistem otomatisasi (Cron Jobs) untuk pengingat pelunasan jamaah yang telah kita kembangkan.

## 3. TAHAPAN PELAKSANAAN
Untuk efisiensi waktu, tim IT akan melakukan langkah sebagai berikut:
1.  **Aktivasi VPS (Prioritas):** Melakukan sewa server terlebih dahulu agar proses instalasi keamanan, sistem backup, dan upload coding bisa diselesaikan segera.
2.  **Registrasi Domain:** Dilakukan setelah server siap. Domain `.co.id` memerlukan verifikasi dokumen legalitas perusahaan.

## 4. PROSEDUR PEMBAYARAN (SISTEM DEPOSIT)
Layanan ini menggunakan sistem **Prepaid Deposit** pada provider IDCloudHost. 
*   **Metode:** Transfer melalui Virtual Account (BCA/Mandiri/BNI).
*   **Faktur:** Sistem akan menerbitkan **E-Invoice (Faktur)** resmi setiap kali pengisian saldo dilakukan, yang dapat digunakan sebagai bukti sah pengeluaran perusahaan.
*   **Keuntungan:** Biaya hanya terpotong saat server aktif, lebih hemat dan transparan.
